#Local_Weather

A Pen created at CodePen.io. You can find this one at http://codepen.io/anusha-medicharla/pen/mEzNoW.
Application created to show the local weather forecast, using HTML, CSS, Bootstrap and AngularJS.
The background of the webpage changes based on the forecast.
Button to view either in Celsius or Farenheit.
Used the "Weather Underground API" for the weather information and parsed the json format resullts.
